# Problem 16: Volumetric Seepage

## Problem Statement
Given an `m x n` integer matrix `heightMap` representing the height of each unit cell in a 2D elevation map, return the volume of water it can trap after raining. Note that water is trapped by boundaries on all four sides.

## Input Format
- A 2D array of integers `heightMap`.

## Example
**Input:** heightMap = [[1, 4, 3, 1, 3, 2], [3, 2, 1, 3, 2, 4], [2, 3, 3, 2, 3, 1]]  
**Output:** 4
