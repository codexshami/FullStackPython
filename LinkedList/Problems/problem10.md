# Problem 10: Cycle Entrance Node

## Problem Statement
Given the `head` of a linked list, return the node where the cycle begins. If there is no cycle, return `None`.

## Input Format
- The `head` of a linked list.

## Output Format
- The node object where the cycle enters, or `None`.

## Example
**Input:** head = [3, 2, 0, -4], pos = 1  
**Output:** Node with value 2  
**Explanation:** The tail connects to node index 1, which has value 2.
