# Problem 20: Optimal Sequence Sorting

## Problem Statement
Given the `head` of a linked list, return the list after sorting it in ascending order. You must solve it in O(N log N) time and O(1) memory complexity (excluding recursion stack).

## Input Format
- The `head` of a linked list.

## Output Format
- The head of the sorted linked list.

## Example
**Input:** head = [4, 2, 1, 3]  
**Output:** [1, 2, 3, 4]
