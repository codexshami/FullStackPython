# Problem 13: Cyclical Displacement

## Problem Statement
Given the `head` of a linked list, rotate the list to the right by `k` places.

## Input Format
- The `head` of a linked list.
- An integer `k`.

## Output Format
- The head of the rotated list.

## Example
**Input:** head = [1, 2, 3, 4, 5], k = 2  
**Output:** [4, 5, 1, 2, 3]  
**Explanation:** 
- Rotate 1: [5, 1, 2, 3, 4]
- Rotate 2: [4, 5, 1, 2, 3]
