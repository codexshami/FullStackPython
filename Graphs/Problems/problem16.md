# Problem 16: The Prim's Web (Prim's Minimum Spanning Tree)

## Problem Statement
Given a connected, undirected, weighted graph, find the Minimum Spanning Tree (MST) using Prim's algorithm. Return the total weight of the MST.

## Input Format
- Number of vertices `V`.
- An adjacency list with weights.

## Example
**Input:** V = 5, edges = [(0,1,2),(0,3,6),(1,2,3),(1,3,8),(1,4,5),(2,4,7),(3,4,9)]  
**Output:** 16
