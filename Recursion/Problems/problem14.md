# Problem 14: The Palindrome Detective (Check Palindrome Recursively)

## Problem Statement
Given a string `s`, determine whether it is a palindrome using recursion. Consider only alphanumeric characters and ignore case.

## Input Format
- A string `s`.

## Example
**Input:** s = "racecar"  
**Output:** True  

**Input:** s = "hello"  
**Output:** False
